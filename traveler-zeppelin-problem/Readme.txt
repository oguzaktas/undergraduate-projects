Gezgin Zeplin Problemi

Programı çalıştırmak için Java SE ve JDK yüklü olması gerekmektedir. Grafik özelliklerini hata almadan çalıştırmak için NetBeans IDE'de (veya çalıştırılmak istenen farklı IDE'de) Swing Application Framework Library eklentisinin kurulu olması ve gerekli API'lerin import edilmiş olması gerekmektedir. Harita jpg formatında resim dosyası olduğu için program dosyalarının bulunduğu dizinde "map.jpg" isminde bulunmalıdır ve haritadaki şehirlerin koordinatlarının bulunduğu "koordinatlar.txt" dosyası yine aynı dizinde bulunmalıdır, yoksa harita çalışmaz.

1) Uygulamayı IDE üzerinden veya işletim sisteminin konsol ekranından gerekli komutları yazarak çalıştırın.
2) İlk olarak başlangıç ve bitiş şehirlerinin girilmesi istenecektir. 1'den 81'e kadar plaka numarası girin.
3) Problem seçme bölümünden 1 (her yolcudan sabit ücret 100 TL alınırsa kaç yolcu ile sefer düzenlenirse max kar elde edilir?) veya 2 (Bir yolcudan kaç TL alınırsa %50 kara ulaşılır?) girin.
4) İstenen çıktılar program dosyalarının bulunduğu dizinde txt formatında ve GUI uygulaması vasıtasıyla harita üzerinden gösterilecektir.

Flying Zeppelin Problem

Java SE and JDK are required to run the program. To run graphical properties without errors, Swing Application Framework Library plugin must be installed in the NetBeans IDE (or the different IDE you want to run) and the required APIs must be imported. The map is an image file in jpg format, so it must be named "map.jpg" in the directory where the program files are located and the "koordinatlar.txt" file in which the coordinates of the cities in the map are located must be in the same directory, otherwise the map will not work.

1) Run the application either through the IDE or by typing the necessary commands on the console screen of your operating system.
2) First you will be asked to enter the starting and ending cities. Enter the plate number from 1 to 81.
3) Enter 1 in the problem selection section (if the fixed price is taken as 100 TL, how many passengers to will get the maximum profit) or 2 (How much price should taken from every passenger to get 50 percent profit?)
4) The desired outputs will be displayed in txt format in the program file's directory and via the GUI application on the map.
